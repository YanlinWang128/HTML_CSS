1. img 标签, 语义: 添加图片
2. 格式: img src="" alt="">
3. 和h标签和p标签不同, img 标签不会独占一行, 没有换行
4. width, 宽度, height, 高度, 属性, 自定义宽高可能导致图片变形
指定宽, 高, 如果没有指定, 则按照默认宽高

等比例拉伸

只指定宽度, 或者高度, 系统会自动等比例计算另一个值, 拉伸
5. title
当鼠标悬停在图片上时, 需要弹出的描述框中显示什么内容 6. alt(alternate) 需要显示的图片找不到时, 显示什么内容, 不可缺少

图片路径 src

相对路径

以和.html文件所在的路径为基准
src = "../panda.jpg" 图片路径在上层路径
src = "image/offer.jpg" 下层路径

绝对路径

每次都从指定盘符开始查找, 一般不使用

br标签, 换行

由于HTML的作用就是给文本添加语义, 而br标签的语义不是另起一个段落换行 而在企业开发中, 一般情况下需要换行都是因为需要另起一个段落, 所以在企业开发中很少使用br
